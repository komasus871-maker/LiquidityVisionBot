# Liquidity Vision v3.5 — Intelligence Core

## Implemented
- Historical Probability Engine based only on completed stored signals.
- Exact setup statistics by setup key, timeframe and direction.
- Similar Setups engine with transparent feature-set similarity.
- New `🧩 Similar Setups` action under every analysis.
- TP1 / TP2 / TP3 / Stop rates, MFE, MAE and reliability labels.

- 100–199: High
