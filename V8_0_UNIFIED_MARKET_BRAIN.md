# LiquidityVisionBot v8.0 — Unified Market Brain

## Core changes
- Decision Brain produces one causal Decision Report for Analyze, Scanner and Watch.
- Expected Value Engine ranks scanner candidates by conservative expected R.
- Market Memory exposes trend/execution evolution and rolling averages.
- Live Intelligence records causal evolution, confidence/probability shift and action changes.
- Unified Decision contract upgraded to v8.0 while preserving legacy output fields.

## Safety
Expected value is explicitly model-based until historical reliability is sufficient. It is a ranking and decision-support estimate, not a guaranteed probability.
