# Liquidity Vision v3.2.1

- READY now requires direction score >= 58, directional edge >= 10, clean blockers and compatible structure/momentum.
- Strong displacement against the proposed direction blocks immediate execution.
- Counter-trend ideas require structural confirmation before READY.
- Balanced markets no longer appear as directional pullback trades.
- Reversal Watch requires multiple reversal clues.
- Pullback ideas can be recorded as WATCHING with status-specific quality thresholds.
- Micro-price formatting supports up to 12 decimals and no longer collapses valid zones to zero.
