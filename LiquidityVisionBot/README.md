LiquidityVisionBot

Trading bot by Noctis.

## v3.3 Intelligence modules

- `📰 News` aggregates RSS headlines, removes duplicates, estimates impact/sentiment and maps affected coins.
- `📊 Dashboard` combines market regime, sentiment, high-impact news, ready setups and personal signal statistics.
- `🧠 Explain Pro 2.0` provides prioritized reasoning, execution plan, invalidation and guarded historical evidence.

RSS feeds can be overridden with the `NEWS_FEEDS` environment variable using comma-separated `NAME|URL` entries.
